# Icons

Place your application icons here:

- `32x32.png` - Windows taskbar icon
- `128x128.png` - General icon
- `128x128@2x.png` - Retina display icon
- `icon.icns` - macOS icon
- `icon.ico` - Windows icon

You can generate these from a single SVG or PNG using tools like:
- https://github.com/tauri-apps/tauricon
- https://www.npmjs.com/package/@tauri-apps/tauricon
