# TypeNext

极简的 Markdown 编辑器，所有操作通过快捷键完成。基于 Tauri 2 + Rust + Vue 3 + CodeMirror 6 构建。

## 核心理念

- **一张白纸**：没有菜单栏、工具栏、侧边栏、状态栏
- **分屏编辑**：左侧编辑区，右侧实时预览
- **全键盘驱动**：所有文件/编辑/导出操作都通过快捷键完成
- **三种输出**：保存为 .md · 导出 PDF · 导出 HTML

## 快捷键

### 文件 & 标签
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` / `Ctrl+O` | 新建 / 打开 |
| `Ctrl+S` / `Ctrl+Shift+S` | 保存 / 另存为 |
| `Ctrl+P` / `Ctrl+Shift+E` | 导出 PDF / 导出 HTML |
| `Ctrl+W` | 关闭标签 |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | 切换标签 |

### 编辑
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` / `Ctrl+Y` | 撤销 / 重做 |

### 视图
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+J` | 切换主题 |
| `Ctrl+H` | 显示帮助 |

## 支持的 Markdown 语法

### 基础格式
| 语法 | 效果 |
|------|------|
| `# ` 到 `###### ` | H1–H6 标题 |
| `**文字**` | 加粗 |
| `*文字*` | 斜体 |
| `~~文字~~` | 删除线 |
| `- ` / `1. ` | 无序 / 有序列表 |
| `> ` | 引用块 |
| `---` | 分割线 |
| `[文字](链接)` | 链接 |
| `![描述](图片)` | 图片 |
| `` `` 文字 `` `` | 行内代码 |
| `` ``` 语言 `` | 代码块（含 mermaid 图表）|

### 增强格式
| 语法 | 效果 |
|------|------|
| `x^2^` / `H~2~O` | 上标 / 下标 |
| `==高亮==` | 高亮文本 |
| `[^1]` ← `[^1]: ...` | 脚注 |
| `$E=mc^2$` / `$$E=mc^2$$` | 行内 / 块级数学公式 |

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Tauri 2 + Rust |
| 前端框架 | Vue 3 + TypeScript |
| 状态管理 | Pinia |
| 编辑器 | CodeMirror 6 |
| Markdown 解析 | remark + rehype |
| 代码高亮 | highlight.js |
| 数学公式 | KaTeX |
| 图表支持 | Mermaid |
| 样式 | SCSS + CSS Variables |

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri:dev

# 构建
npm run tauri:build
```
